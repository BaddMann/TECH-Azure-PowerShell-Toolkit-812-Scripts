Getting Started

Complete setup guide to get you running Azure automation scripts in under 5 minutes. This guide covers installation, authentication, and running your first script.

1 Prerequisites & Installation

Before You Begin

  • Azure Subscription: Active Azure subscription with appropriate permissions
  • PowerShell 7.0+: Cross-platform PowerShell (recommended for best compatibility)
  • Git: For cloning the repository (optional - you can download ZIP instead)
  • Text Editor: VS Code, PowerShell ISE, or any preferred editor

Windows Installation

PowerShell (Run as Administrator) 
# Install PowerShell 7 (if not already installed)
winget install Microsoft.PowerShell

# Install Azure PowerShell module
Install-Module -Name Az -Repository PSGallery -Force -Scope CurrentUser

# Verify installation
Get-InstalledModule -Name Az
$PSVersionTable.PSVersion

macOS Installation

Terminal 
# Install PowerShell 7 using Homebrew
brew install --cask powershell

# Start PowerShell
pwsh

# Install Azure PowerShell module
Install-Module -Name Az -Repository PSGallery -Force -Scope CurrentUser

# Verify installation
Get-InstalledModule -Name Az
$PSVersionTable.PSVersion

Linux Installation (Ubuntu/Debian)

Bash Terminal 
# Update package index
sudo apt-get update

# Install prerequisites
sudo apt-get install -y wget apt-transport-https software-properties-common

# Download and install PowerShell
wget -q https://packages.microsoft.com/config/ubuntu/20.04/packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
sudo apt-get update
sudo apt-get install -y powershell

# Start PowerShell
pwsh

# Install Azure PowerShell module
Install-Module -Name Az -Repository PSGallery -Force -Scope CurrentUser
Alternative: You can also use Azure Cloud Shell which has PowerShell and all Azure modules pre-installed. Access it at shell.azure.com.

2 Download the Toolkit

Choose your preferred method to download the Azure Enterprise Toolkit:

Option 1: Clone with Git (Recommended)
Terminal/PowerShell 
# Clone the repository
git clone https://github.com/wesellis/Azure-Enterprise-Toolkit.git

# Navigate to the automation scripts
cd Azure-Enterprise-Toolkit/automation-scripts

# List available categories
Get-ChildItem -Directory
Option 2: Download ZIP Archive

Download the latest release from GitHub:

  1. Visit GitHub Repository
  2. Click the green "Code" button
  3. Select "Download ZIP"
  4. Extract the archive to your preferred location
Pro Tip: Using Git clone allows you to easily pull updates with git pull to get the latest scripts and improvements.

3 Azure Authentication

Before running any scripts, you need to authenticate to your Azure subscription. Choose the method that best fits your scenario:

Interactive Login (Recommended for Getting Started)
PowerShell 
# Connect to Azure with interactive login
Connect-AzAccount

# Verify connection and show current subscription
Get-AzContext

# List all available subscriptions
Get-AzSubscription

# Switch to a specific subscription (if needed)
Set-AzContext -SubscriptionId "your-subscription-id"
Service Principal Authentication (Production/Automation)
PowerShell 
# Create credential object
$credential = Get-Credential

# Connect using service principal
Connect-AzAccount -ServicePrincipal -Credential $credential -Tenant "your-tenant-id"

# Or using certificate-based authentication
Connect-AzAccount -ServicePrincipal -CertificateThumbprint "cert-thumbprint" -ApplicationId "app-id" -Tenant "tenant-id"
Security Note: For production environments, always use service principals with appropriate RBAC permissions. Avoid using personal accounts for automated processes.

4 Run Your First Script

Let's start with a simple script to list your Azure resources and verify everything is working correctly:

PowerShell - First Script Example 
# Navigate to the toolkit directory
cd Azure-Enterprise-Toolkit/automation-scripts

# Run a simple resource listing script
.\General-Utilities\Azure-Resource-Inventory-Tool.ps1 -Action "List" -ResourceType "All"

# Or try a VM management script
.\Compute-Management\Azure-VM-Management-Tool.ps1 -ResourceGroupName "your-rg-name" -Action "List"

# For help with any script, use Get-Help
Get-Help .\Compute-Management\Azure-VM-Management-Tool.ps1 -Detailed

Verification Steps

  1. Check Output: You should see a list of your Azure resources
  2. No Errors: Scripts should run without authentication or module errors
  3. Proper Formatting: Output should be well-formatted and readable
  4. Log Files: Check for any log files created in the script directory
Script Parameters: All scripts include comprehensive help documentation. Use Get-Help script-name.ps1 -Examples to see usage examples.
