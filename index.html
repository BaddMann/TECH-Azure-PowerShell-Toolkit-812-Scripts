Quick Start Guide

Get started with Azure automation in under 5 minutes with our step-by-step guide.

1

Install Prerequisites

Install PowerShell 7.0+ and the Azure PowerShell module on your system.

2

Authenticate to Azure

Connect to your Azure subscription using Connect-AzAccount or service principal authentication.

3

Clone Repository

Download the toolkit from GitHub and explore the organized script categories.

4

Run Your First Script

Execute scripts with proper parameters and customize for your environment needs.

# Install Azure PowerShell Install-Module -Name Az -Repository PSGallery -Force # Connect to Azure Connect-AzAccount # Clone the repository git clone https://github.com/wesellis/Azure-Enterprise-Toolkit.git # Navigate to scripts cd Azure-Enterprise-Toolkit/automation-scripts # Run a sample script .\Compute-Management\Azure-VM-Management-Tool.ps1 ` -ResourceGroupName "my-rg" ` -Action "List"
